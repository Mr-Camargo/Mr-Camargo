When I was on elementary school, I started gaining curiosity for computer science and making videogames. I eventually started learning basic Javascript algorithms with the help of Scratch, a drag-and-drop visual coding website where you can build and share games, stories and animations.

I started using it on the classroom, and I was able to learn the fundamentals of coding in a way that it could be interactive and fun for me as a child.

Example project: Poyo Bomber

Poyo Bomber is a Scratch-based game in which your objective is to help Kirb collect 13 candies using the arrows on your keyboard, while avoiding the bombs that will appear along the way.

This game was made when I was in 5th grade, and finished it on June 10th, 2020 as it says on the project page.
